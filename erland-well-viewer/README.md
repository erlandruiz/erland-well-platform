# Erland Well Viewer

**Español:**
Aplicación visual en React para ingresar, calcular y graficar surveys de pozos direccionales usando la librería privada `erland-well-core`.

**English:**
React visual application for entering, calculating, and plotting directional well surveys using the private `erland-well-core` library.

---

## Estado del proyecto / Project Status

**Español:**
Primera versión visual funcional en desarrollo. Actualmente permite ingresar surveys, seleccionar unidades, calcular resultados completos, visualizar trayectoria en 2D/3D y cargar pozos modelo para demostración.

**English:**
First functional visual version in development. It currently allows entering surveys, selecting units, calculating full results, visualizing trajectory in 2D/3D, and loading demo wells.

```txt
Version: 0.1.0
```

## Objetivo / Goal

**Español:**
Crear una aplicación visual tipo visor técnico para pozos direccionales, capaz de ingresar datos de survey, calcular trayectoria usando Minimum Curvature y mostrar resultados en tablas y gráficas profesionales.

**English:**
Create a technical visual viewer for directional wells, capable of entering survey data, calculating trajectory using Minimum Curvature, and displaying results in professional tables and charts.

### Nuevo proyecto / New Project

**Español:**
El botón `Nuevo proyecto` limpia los datos guardados y reinicia el visor.

**English:**
The `Nuevo proyecto` button clears the saved data and resets the viewer.
Before deleting the data, the application shows a confirmation message.

Al reiniciar / On reset:

* Se borra el survey actual / The current survey is cleared.
* Se vuelve a la estación inicial MD 0, INC 0, AZI 0 / It returns to the initial station MD 0, INC 0, AZI 0.
* Se desbloquean las unidades / Units are unlocked.
* Se limpia el `localStorage` / `localStorage` is cleared.

## Flujo de uso / Usage Flow

**Español:**

1. Seleccionar unidades: Feet o Meters.
2. Ingresar surveys en la vista `Entrada Survey`, o cargar un pozo modelo desde `Demo wells`.
3. Revisar cálculos completos en `Resultados`.
4. Ver la trayectoria horizontal en `Vista Planta 2D`.
5. Configurar `Vertical Section Direction` y revisar `Vista Vertical 2D`.
6. Revisar la trayectoria espacial en `Vista 3D`.
7. Usar `Nuevo proyecto` para limpiar todo y empezar nuevamente.

**English:**

1. Select units: Feet or Meters.
2. Enter surveys in the `Survey Input` view, or load a demo well from `Demo wells`.
3. Review full calculations in `Results`.
4. View the horizontal trajectory in `2D Plan View`.
5. Configure `Vertical Section Direction` and review `2D Vertical View`.
6. Review the spatial trajectory in `3D View`.
7. Use `New Project` to clear everything and start again.
